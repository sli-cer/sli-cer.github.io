Python-GUI实战之TKinter

发表于 2021-06-06 阅读次数：
本文字数： 8.4k 阅读时长 ≈ 8 分钟

一、缘起

终于有时间记录一下了。这次是年前的一个项目，过年前后开始需要用到GUI，我负责的整个GUI部分的编写，下面详细介绍一下当中用到的GUI编程技术。

二、需求分析

这个项目的需求比较简单，两个页面就覆盖了全部的内容，tab页切换和菜单切换都实现了。主要功能是两个，计算和展示：计算功能需要几个输入，点击按钮在固定位置展示计算所得的数值；展示功能是从文件中读入数据（类似关系型数据库格式），然后按表格形式展示，旁边有输入框控制筛选条件，点击筛选按钮可刷新展示数据。另外计算功能的展示还用到柱状图集成进GUI。大概就这些。

三、技术介绍

这里就介绍一下用到的框架如TKinter、matplotlib、pandas、pyinstaller及其在软件中的作用。

1、TKinter

TKinter是python的标准TK GUI工具包的接口，安装python自带，功能也比较强大，基本满足大多数GUI软件的需求场景，相比wxPython和pyQt等框架节省了安装步骤，也更轻量；但无法定制样式，布局的方式也有些僵化，在需求更加复杂的场景还是应该使用C++Qt。

2、matplotlib

matplotlib是一个python的2D绘图库，优点是跨平台和高质量。这里将matplotlib集成进TKinter的canvas里，实现柱状图的展示。

3、pandas

pandas是基于numpy的用于处理数据的工具，优点是能够处理大型数据和高效。这里用于处理读入的数据，将其做格式化和筛选，使用起来还是很方便的，学习成本也不高。

4、pyinstalller

pyinstaller是一个将python程序打包成独立可执行程序的软件包，可跨平台，用户无需安装python环境。
